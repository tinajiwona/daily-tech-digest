## 💰 宏观与政策

[美国将阿里巴巴和比亚迪加入涉军企业名单](https://www.ftchinese.com/story/001109010)  
**政策内容 + 背景 + 影响分析**: 美国财政部最新决定将两家中国科技和制造巨头列入涉军企业名单，限制美国投资者持有相关证券。此举是美国对华科技限制的进一步升级，背景是中美战略竞争加剧。对普通投资者而言，需警惕相关中概股波动风险，建议分散投资，关注不受地缘政治影响的优质标的。

[丹麦首相：欧洲需要有"紧迫感"以求生存](https://www.ftchinese.com/story/001109007)  
**政策内容 + 背景 + 影响分析**: 丹麦首相在达沃斯论坛上呼吁欧洲加快经济改革，应对全球竞争压力。欧洲面临人口老龄化、能源转型缓慢等挑战，经济增长乏力。这一表态暗示欧洲可能采取更积极的财政和产业政策。对国内投资者而言，中欧贸易摩擦可能加剧，但欧洲绿色转型也为新能源、环保产业带来机遇。

[央行官员异议告诉我们什么？](https://www.ftchinese.com/story/001109006)  
**政策内容 + 背景 + 影响分析**: 央行内部出现政策分歧，反映当前经济决策面临复杂局面。一方面要稳增长，另一方面要防通胀，政策平衡难度加大。这种分歧可能预示未来货币政策将更加灵活。对普通人而言，房贷利率可能保持稳定，存款利率仍有下调空间，建议合理配置资产，适当增加债券类投资。

**行业动态 + 数据支撑 + 影响分析**: 全球AI革命正在重塑高等教育体系，英美两国采取不同应对策略。美国高校与企业深度合作，英国则注重基础学科与AI融合。数据显示，AI相关岗位需求年增40%，但毕业生供给不足。这一趋势将影响未来人才市场格局，对投资者而言，教育科技、职业培训板块长期看好，建议关注AI教育解决方案提供商。

[小鹏为什么这么"烦"L3](http://www.huxiu.com/article/4835440.html?f=wangzhan)  
**行业动态 + 数据支撑 + 影响分析**: 小鹏汽车在自动驾驶L3技术上投入巨大但商业化进程缓慢，反映智能驾驶落地面临技术、法规等多重挑战。数据显示，2025年L2级渗透率达35%，但L3不足1%。车企在技术投入与商业回报间寻求平衡，对消费者而言，短期内L2+仍是主流，建议关注具备数据优势的车企。

[中国车企，要接受新车销量"仅微涨"的新常态](http://www.huxiu.com/article/4833852.html?f=wangzhan)  
**行业动态 + 数据支撑 + 影响分析**: 中国汽车市场进入存量竞争阶段，2025年销量增速降至3%，远低于过去十年平均15%的增长。行业从增量市场转向存量市场，竞争焦点从规模转向质量。对投资者而言，建议关注具备技术壁垒和品牌溢价的车企，以及受益于存量市场的汽车后市场服务企业。
